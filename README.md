# radio-data-collection

无线电硬件与数据采集组代码仓库

\# 无线电数据采集工具包



\## 项目简介

基于Python的无线电设备参数配置、信号采集模拟和IQ信号处理工具包。



\## 功能特性

\- 📊 设备参数配置管理（JSON格式）

\- 📡 信号采集模拟

\- 🔬 IQ信号处理与分析

\- 🧪 完整测试流程

\- 🔄 Git协作支持



\## 快速开始



\### 安装依赖


pip install -r requirements.txt

\### 运行测试
bash
python radio_basic_toolkit.py
\### 使用示例
bash
python example_usage.py
\### 项目结构
text
